          # 在windows安装python

          在python官网下载适用于windows x64 3.5的安装包。

          • 下载完成后，点击安装包开始安装。

          需要勾选Add Python 3.5 to PATH

          • 如果安装成功，就会提示

          • 打开cmd终端，输入python -V验证

          • 配置阿里云镜像源，这样下载库会更快

          windows系统在新建

          C:\Users\Administrator\pip\pip.ini
          1

          写入下面的内容

          [global]
index-url = https://mirrors.aliyun.com/pypi/simple/

[install]
trusted-host=mirrors.aliyun.com

          2
          3

          • 安装esptool工具
          pip install esptool
          1

          安装成功，就会提示下面的内容

          # 下载固件

          在官方网站下载esp8266 micropython固件

          要选择Firmware for ESP8266 boards。这里下载最新版的固件

          # 安装CP2102驱动

          我这块版，用的是CP2102芯片。所以安装CP2102驱动。

          • 在官方网站下载CP210X 下载用于 Windows 10 通用 (v10.1.8) 的版本的驱动。

          • 选择x64位的安装程序

          驱动安装成功后，就能在windows 设备过滤器看到设备

          需要注意串口号，比如COM3，刷固件的时候，会用到。

          如果看不到设备，可能需要更换一条数据线

          # 开始刷固件

          打开终端，输入下面的命令，开始刷固件

          esptool.py --port COM3 --baud 460800 write_flash --flash_size=detect -fm dio 0 esp8266-20191220-v1.12.bin
